# Lista de Compras en React.js

## Descripción
Este proyecto es una aplicación web básica desarrollada con React.js. Su función es permitir al usuario crear y administrar una lista de compras de manera sencilla e interactiva.

La aplicación permite:
- agregar productos,
- marcarlos como comprados,
- eliminarlos de la lista.

## Objetivo
El objetivo de este proyecto es resolver un problema cotidiano: organizar productos de compra de forma rápida y visual. Además, sirve para aplicar los conceptos básicos de React.js en una aplicación funcional.

## Problema que resuelve
Muchas veces las personas olvidan productos cuando van de compras o no llevan un control claro de lo que ya adquirieron. Esta aplicación ayuda a organizar mejor esos productos en una lista simple y fácil de usar.

## Tecnologías utilizadas
- React.js
- JavaScript
- CSS
- Vite

## Funcionalidades
- Agregar productos a una lista
- Marcar productos como comprados
- Eliminar productos
- Interfaz sencilla y responsiva

Cómo ejecutar el proyecto

Crear el proyecto con Vite:
npm create vite@latest lista-compras-react

Seleccionar las opciones:
Framework: React
Variant: JavaScript

Entrar a la carpeta del proyecto:
cd lista-compras-react

Instalar dependencias:
npm install

Ejecutar el proyecto:
npm run dev

Abrir en el navegador la dirección que aparece en la terminal, normalmente:
http://localhost:5173/
